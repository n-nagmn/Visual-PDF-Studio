# Visual PDF Studio

ブラウザ完結で動作する、高機能かつ完全ノーログのPDF編集・変換ツールです。サーバーへのファイルアップロードを一切行わず、すべての処理をクライアントサイドのJavaScriptで安全に実行します。

## 特徴

* **完全ノーログかつオフライン動作**: 読み込んだPDFや画像データは外部に送信されません。機密文書も安全に処理できます。
* **13種類の強力なツール**:
  * **結合・分割・整理**: 複数のPDFを1つにマージ、不要なページの削除、ドラッグ＆ドロップによるページ順の入れ替え。
  * **回転**: ページ単位、または全ページの向きを一括修正。
  * **セキュリティ**: パスワードによる閲覧制限の追加、および制限の解除。
  * **装飾・追記**: 透かし（ウォーターマーク）の追加、ページ番号の付与、テキストの追記。
  * **変換**: JPG/PNG画像からPDFへの変換、PDFからJPG画像への書き出し、PDFからのテキストデータ抽出。
  * **その他**: 空白ページの挿入、タイトルや作成者などのメタデータ編集。

## 使用技術とライブラリ

* HTML / CSS / Vanilla JavaScript
* [pdf-lib](https://pdf-lib.js.org/): PDFの生成・編集・メタデータ操作
* [PDF.js](https://mozilla.github.io/pdf.js/): PDFのレンダリング・テキスト抽出
* [SortableJS](https://sortablejs.github.io/Sortable/): ドラッグ＆ドロップによるUI操作
* FontAwesome: アイコン表示
